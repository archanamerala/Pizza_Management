# Pizza Management System In Python 

## Description
The Pizza Management System is a GUI-based desktop application written in Python. It allows users to register, log in, select pizzas with different sizes and toppings, add items to a cart, and place an order by providing delivery details (city, building, locality, house number, contact number). The system includes both an admin side and a user side, but **only the user side is functional** – the admin section is not implemented.

---

## Features

### User Side (Fully Working)
- User registration and login
- Select pizza with available sizes
- Choose toppings
- Enter quantity
- Add to cart
- Place order with delivery address and contact number
- Simple GUI – easy to navigate

### Admin Side (Not Functional)
- Login area exists but no further functionality
